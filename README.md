# stats220

This is my repo for **STATS 220**.
1. Learn *R programming* and the *tidyverse* for data science
2. Work with data: generate, import, clean, and visualize data
3. Learn how to store data and work with different file formats
4. Use modern data sources and data technologies
5. Use data to support decision-making and solve problems in business and research
6. Develop creative thinking with data tools
7. Learn how to communicate data science results effectively

## A little about me:

- I am undertaking a degree in Statistics at the University of Auckland.
- I am taking STATS 220 because it is an advanced course following STATS 125 and is required for my degree. It helps me build on the foundational knowledge I gained in 125 and develop more advanced data analysis skills.
- I am interested in learning about how to use programming tools to analyse and communicate data effectively.
